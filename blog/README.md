# Blog Posts

Add new articles as `.md` files. Use the filename as the URL slug (e.g. `2025-02-13-my-post.md` → `/blog/2025-02-13-my-post`).

## Frontmatter (required)

```yaml
---
title: "Your Post Title"
description: "Short description for SEO and listings"
date: "2025-02-13"
tags: ["Tag1", "Tag2"]
---

Optional: category, thumbnail
```

Then write your content in Markdown below the frontmatter.
